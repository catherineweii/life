我可以亲你一下吗

2016-06-03

03/06/2016

ca：哎呀，你干嘛这么萌。我想起来我的缺点了

ck：喜欢你藏不住了。我的缺点就是特别黏，但是又没什么话

ca：我喜欢被黏哈哈哈哈哈哈哈

ck：不过你告诉我缺点也没用，因为我现在是盲目的，视力最差的时候。

…

ck：可以亲你一下吗！

ca：现在你亲不到呀

ck：刚刚准备说什么呢，那你答应就是嘛

…

ck：好喜欢听你说话啊可以再说一句吗

ca：“说什么呀”

ck：满足了已经！你怎么这样，我睡不着了的

…

ck：我是一个很无聊的人，这可能是我最大的缺点了

ca：会唱歌会撒娇，多好嘛

ck：这些都会腻的啦

ca：平平淡淡才是真啊

…

ca：哎呀感觉每天都跟你说不够话

ck：因为，你喜欢我！

ca：我可没有说喜欢你

ck：我喜欢你哦，超喜欢

ca：哈哈 所以你要跟我说上一一世情话嘛

ck：要。可是其实我翻来覆去也就几句，你要是听不腻的话我就一直跟你说

…

ck：好饿，麻烦给我做早餐

ca：起床去吃饭嘛

ck：谢谢，不是你做的就不吃

你跟我说让我起来去吃东西好不好

ca：先去洗漱

ck：不行！你要说话，语音，语音！

…

ca：是不是快要去上课啦

ck：没呢

想

见

想

见

想

见

我要是想一万遍能不能见到你呀

ca：大概会

ck：那你也要想一万遍好不好

ca：好嘛

…

ca：下午回家自己做高铁吗

ck：和朋友一起同一趟高铁，她去武汉，我去咸宁。会吃醋吗！

ca：祝儿孙满堂

ck：我要做你孩子的父亲，好不好

…

ck：有时候想想他们好可怜，有时又觉得不能这样想

ca：我跟你一样哈哈

ck：在一起在一起

…

ck：刚刚想说去洗个澡，但是又不想现在去了。想再跟你说一会儿话

ca：去洗个澡，然后舒舒服服的躺在床上和我说

ck：再说五分钟！

ca：好

ck：我从告诉你我喜欢你开始 就越来越觉得自己好差好差好差 你会不喜欢我的。总会担心做的不好就会失去你，那天也是很伤感的写了那个故事

ca：我觉得哥哥很酷很酷～你会看正午恶魔我就很吃惊了，而且个个还去英国学习对不对 ，总觉得那里好像很难呀，然后你都完成了耶。而且反正你就是很关心我就对了，我可以隔着太平洋感受到你喜欢我的能量光圈

04/06/2016

ck：我刚刚在想要给你录一首歌

ca：什么歌呀

ck：不过还没想好什么歌呢，一首歌。一定要很甜的一首。我得精挑细选！

ca：嗯！

ck：不然我心里的甜都要溢出来了

ca：哎呀，太会撒娇了

5/06/2016

ck：我想你所有的伤心难过痛苦都能有我的一份力量在支持。这不是我想哄你的情话，这是我真的想做到的事情。

ca：哈哈 你这么说我可要泪崩了

ck：要笑！我乖乖最爱笑了

06/06/2016

ca：我要从大山上给你采来欢乐的花，那喇叭藤花。那褐色的榛子，那装满了亲吻的野藤花篮。我要在你身上去做，春天在樱桃树上做的事情。

07/06/2016

ck：我在计划改火车票回北京见你

ca：真的咩….

ck：嗯。我就是，想见你。

…

ca：你在干嘛呀

ck：刚刚洗了头发。一直都没干什么，做几个俯卧撑床上趴一会儿对你傻笑一会儿。就到现在了

08/06/2016

ca：我觉得我离不开你了怎么办

ck：有一个办法，那就是不离开我。我会努力的！真的，我会努力的！

…

ck：问你一个问题好不好～

ca：好

ck：你能不能感觉到我喜欢你呀

ca：能呀。我不太想开始，我总觉得开始了就好像要失去你了，我害怕你不喜欢我。而且我情绪我自己都受不了。

ck：那我们不着急，先让我追你个十年八年的。相信我，其实我也害怕。我害怕会结束，怕你不喜欢我，怕我还是一样幼稚。然后你会发现我很幼稚然后腻了。

ca：不腻呀

ck：但是我会怕，就算你告诉我我还是会怕

ca：怕就好好做嘛，好好珍惜

ck：不过我会努力跟你把现在过得开心

09/06/2016

ck：我在坐客车去温泉，热死了。

ca：哈哈，脱光！

ck：不脱光给别人看

ca：哈哈哈哈哈哈，那给谁看呀～

ck：哎呀不好意思说也不知道人家会不会很嫌弃

ca：不知道你在说谁耶

ck：是… 我在追的一个小姑娘呐

ca：原来是这样呀，加油哦

ck：嗯！已经做好了追十年的打算呢！

ca：那到时候你老了怎么办呀

ck：她可能就不喜欢了，好难过。那我快点追上！

ca：哈哈哈哈 你加油，肯定会成功的

ck：好！谢谢你的鼓励！等我成功了就跟你结婚！

…

ca：在想什么

ck：在想你。想跟你在不热的天气牵手散步

09/06/2016

ck：自己刚刚准备跟你啰嗦吃东西的事情

ca：啰嗦呀

ck：还是不要啰嗦，乖乖你好好吃就行。肯定知道照顾自己的，我就提醒你吃药好了

ca：你都懒得说我了耶

ck：傻瓜我是觉得会太啰嗦了。才不是懒，你的事情我全部都要管。
